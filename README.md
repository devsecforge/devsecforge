## 📅 2025 Roadmap · Anchor Repositories

2025 is about **turning ideas into structured, documented, reusable systems**. Three repositories anchor the roadmap and get the deepest, most iterative treatment throughout the year.

### `2025-secure-ci-cd-lab`
- **What:** A complete CI/CD pipeline with built-in security checks.
- **Why:** To show how DevSecOps can be practical, repeatable, and production-ready.
- **How:** GitHub Actions / GitLab CI, IaC scanning, policy-as-code, container security.
- **When:** Iterated throughout 2025 with tagged releases and changelogs.

### `2025-k8s-security-blueprints`
- **What:** Kubernetes security patterns, manifests, and hardening guides.
- **Why:** To help teams secure clusters without guesswork.
- **How:** Network policies, RBAC, PodSecurity, admission controllers, logging & monitoring.
- **When:** Built step by step with examples and real-world scenarios.

### `2025-ai-threat-modeling`
- **What:** Using AI to support threat modeling and risk analysis.
- **Why:** To augment human judgment with intelligent assistance.
- **How:** Prompt libraries, workflows, sample models, and documentation.
- **When:** Released in phases with use cases and demo scripts.
